# 本文记录服务器启动方式

## 0. 环境配置

服务器上启动之前，请先准备好相应的环境配置，请查看 [安装环境](安装环境.md) 文档进行服务器环境初始化

## 1. 源码方式构建

进入工作目录，下来最新代码

```bash
cd /home/admin/workspace

git clone git@github.com:liuyueyi/quick-forum.git
```

接下来就是编译启动

```bash
cd quick-forum
./launch.sh start
```

提示：

- 若launch.sh脚本没有执行权限，可以通过命令行 `chmod +x launch.sh` 添加
